# J70 jubilæumsskrift

Vi laver en gennemgang af TÅGEKAMMERETs historie siden 1956, hvor foreningen blev stiftet.

## Struktur

Skriftet skrives i LaTeX, og hver undersektion skrives i egen fil, som så inkluderes i en central main-fil.

Arbejdet fordeles via GitHub Issues vha. først-til-mølle.

main-branchen er beskyttet, og der skal derfor laves PR til alle ændringer. Husk at oprette nye branches på hvert seperat issue, så alt hænger sammen.
